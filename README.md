# CULTURA_DIGITAL_3RO_A_JHON_BUSTAMANTE
Actividades de la clases

1.- El código implementa un detector de palíndromos en Python que primero limpia el texto ingresado eliminando espacios, caracteres especiales y convirtiendo todo a minúsculas, luego compara la cadena resultante con su versión invertida para determinar si es un palíndromo; el programa ejecuta automáticamente 8 ejemplos predefinidos (palabras, frases y números) y después entra en un modo interactivo donde el usuario puede ingresar sus propias palabras o frases hasta escribir "salir" para terminar.
